# TxtUrl

A minimalist text editor fully embedded in the URL - no backend, no storage, no account.

Write your text, share the link. Everything is encoded directly in the URL hash.

No server, no database, no tracking, no traces: the text never leaves your browser.

## Features

- Markdown rendering
- QR code generation
- Print to `.pdf`
- Download as `.md`

## Markdown syntax

````md
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
Normal text

---

> This is a blockquote

- unordered item
* another item

1. ordered item
2. another item

```
inline code
```
````
